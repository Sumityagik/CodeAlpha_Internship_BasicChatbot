# CodeAlpha_Internship_BasicChatbot
# Project 2: Rule-Based Chatbot
A lightweight, rule-based conversational agent designed to interact with users through predefined triggers and structured text matching.

# Features
Pattern Matching: Standardizes human input by stripping whitespace and converting text to lowercase to ensure robust match execution.

Dynamic Interactivity: Continues a seamless conversation loop until the user explicitly triggers an exit phrase (e.g., "bye").

Fallback Logic: Provides a courteous, informative response when user inputs fall outside the predefined rule parameters.

# Key Concepts Utilized
. Clean code structuring using modular Python functions.

. String manipulation and normalization methods (.lower(), .strip()).

. Infinite loop structures (while True) with programmatic break conditions.
